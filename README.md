Asteroids
=========

A HTML 5 & WebSockets online multiplayer asteroids game built with:
 - SignalR / MVC
 - JavaScript
 - Two.js
 - Jquery
 - Hands
 - Eyes
 - Tongue

Cross platform and lots of fun.


Working example: http://asteroids.azurewebsites.net/
